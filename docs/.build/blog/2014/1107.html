20141107 (Friday, 07 November 2014)¶

I recently had modified ml.beid to add a new user_group. This change turned out yesterday to cause a regression in CPAS de Châtelet: the “Read eID card” button had disappeared for everybody.

Komiya Takeshi wrote a survey of Sphinx extensions. This inspired me to work a bit on atelier.sphinxconf:

  • atelier.sphinxconf.blog no longer requires a DJANGO_SETTINGS_MODULE to be defined. This was due to a stupid dependency for the month_name function.
  • moved two directives from atelier.sphinxconf.insert_input to a separate module atelier.sphinxconf.complex_tables.

One day I should move them out of atelier into a series of sphinx-contrib projects...
