20140119 (Sunday, 19 January 2014)¶

I am now a fellow¶

Last week I did something which I had been thinking of for a long time: I registered as a “fellowship” member of the Free Software Foundation Europe.

If in the future I should happen to restart blogging about things that are not directly related to Lino, then it will be in Luc’s blog.

Upgraded demo sites on lino-framework.org¶

Between 16.20 and 16.30 the demo databases on lino-framework.org were down because I did a git pull and a initdb_demo.

Note that I did this in the Tallinn airport while waiting for my plane to Brussels, because a second Internal Server Error had come in. Somebody obviously trying Lino Così. Now it works.
