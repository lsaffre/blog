20140228 (Friday, 28 February 2014)¶

Lino Faggio¶

Started to update Changes in Lino-Faggio.

Added new virtual field linked_date (“Date”) in EventsByCourse. This makes the date of each event clickable, and clicking it opens a table with all events on that day.

(Unofficial) release of Lino Faggio version 0.0.3 for Roger.

Lino Welfare¶

New app module lino.modlib.sepa. Using this app requires Ben Konrath’s django-iban module. Note that I did not declare this dependency for Lino (the framework) since many Lino user’s won’t need it. But I did add it to the install_requires of Lino Welfare.

The “Aids” tab of lino_welfare.modlib.pcsw.PersonDetail now shows the slave table sepa.AccountsByPartner. Removed the two bank account fields.

The managed checkbox field is a specific addition by Lino Welfare. It indicates whether a client account is being “managed” by the social assistant.

I discovered that some files in Lino Welfare had “This file is part of the Lino-Faggio project.” in their copyright header. Replaced these by “This file is part of the Lino Welfare project.”
