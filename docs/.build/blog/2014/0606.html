Friday, June 6, 2014¶

Estonian places¶

The new package lino.projects.estref is the Estonian equivalent of The Belgian Reference. But without the alternative plain user interface.

I used it mainly to test lino.modlib.countries.fixtures.est which is now a proof of concept for my vision of using The Python serializer as data format for maintaining the list of places of a country.

More documentation¶

  • I discovered and fixed the documentation bug in Installing Lino Così which was probably the main reason why Milton asked Lino Cosi app installation
  • And I started /dev/translate/index, also dedicated to Milton.

And note that you must pull the newest version of Lino before trying it, because there were some minor things missing, e.g. a Spanish demo user “Rodrigo Rosalez”.
