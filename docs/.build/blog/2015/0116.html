Friday, January 16, 2015¶

Still working on the docs¶

Continued to change the main page of /dev/index. Instead of just listing the titles of subsections, I add a summary of every section.

No more need to set user_model¶

The lino.modlib.users plugin now automatically sets user_model.

Until now, application developers had to set user_model to 'users.User' and add lino.modlib.users to their lino.core.site.Site.get_installed_apps() in order to enable user management.

To make the implementation more clean, I added new methods lino.core.plugin.Plugin.on_init() and lino.core.site.Site.set_user_model().
