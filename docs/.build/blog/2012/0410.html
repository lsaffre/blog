20120410¶

list index out of range¶

name2kw and street2kw now both return an empty dict when given an empty string. Until now, street2kw() returned {'street':''} and name2kw caused an IndexError “list index out of range”.

Installation¶

Lino now creates automatically the following symbolic links if they don’t exist:

  • /media/lino/ : Lino’s :srcref`/media` directory
  • /media/extjs/ : ExtJS library (lino.Lino.extjs_root)
  • /media/extensible/ : Ext.ensible library (lino.Lino.extensible_root)
  • /media/tinymce/ : TinyMCE library (lino.Lino.tinymce_root)
