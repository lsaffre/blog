20120229¶

The “Sandbox étude” started a few days ago was still haunting me. Opened ticket docs/tickets/60 and understood that the existing lino.modlib.contacts is in fact almost perfect, except that some names are wrong. We should one day rename “Contact” to “Partner”, “Role” to “Contact” and “RoleType” to “Role”.

Posted a question about plain css tooltips (abandoned a few days ago) to the ExtJS4 forum.

Also re-activated the only-half-working css overrides necessary for use_css_tooltips in /media/extjs/lino.css since they did not cause yesterday’s layout problem.

Und noch ein kleiner Geistesblitz zwischendurch: Slave-Grids müssen ja im Insert-Fenster disabled sein, weil man die erst nach dem ersten Speichern eingeben darf (wenn der Record erstellt worden ist). Störend war nur, dass diese Slave-Tabellen zwar disabled waren, aber oft irgendwelche Daten enthielten von früheren Aufrufen enthielten. Lösung: Jetzt sind sie nicht mehr disabled, sondern hidden.
