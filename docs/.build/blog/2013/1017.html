20131017 (Thursday, 17 October 2013)¶

Cool, Lino has two new features (needed for Lino Polly but probably useful for many other projects):

  • The clickable “-1”, “0” and “+1” links are a usage example for the new put_button method.

  • Actor has a new overrideable class method get_detail_title. This was used to replace the window title

    “My responses » Robin Rood’s response to Matthew 1:17-25”

    by

    “My responses » My response to Matthew 1:17-25”
