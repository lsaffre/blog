Custom fields of Choicelist Choices¶

Something cool: application code can now simply define custom fields of a lino.core.choicelists.Choice by declaring them on the lino.core.choicelists.Choicelist. First usage example are cal.EventStates and lino.UserLevel. If you wanted to show such custom fields in a table, then you had to define a virtual field:

@fields.virtualfield(models.CharField(_("Short name"),max_length=2,
    help_text="used to fill UserProfiles"))
def short_name(cls,choice,ar):
    return choice.short_name

Now you can just write:

short_name = models.CharField(_("Short name"),max_length=2,
    help_text="used to fill UserProfiles")
