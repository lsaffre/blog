What does the following git message mean?

::

  From gitlab.com:lino-framework/xl
   * branch              master     -> FETCH_HEAD

For example::

  $ go xl
  $ git pull
  From gitlab.com:lino-framework/xl
   * branch              master     -> FETCH_HEAD
  Already up to date.
  $ go ug
  $ git pull
  Already up to date.
